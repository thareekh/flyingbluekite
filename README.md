# Flying Blue Kite Website

A modern, static website conversion of the Flying Blue Kite blog, built for GitHub Pages.

## Features
- Modern dark/light mode toggle
- Responsive design
- AI Story Generator (creates daily posts based on your style)
- Instagram integration (@thareeekh)
- Reading progress indicator
- SEO optimized
- No em dashes (all converted to proper punctuation)

## Setup Instructions

1. Create a new repository on GitHub named `flyingbluekite.github.io`
2. Upload all these files to the repository
3. Replace placeholder images in `/assets/images/` with your actual blog images
4. Update `_config.yml` with your email
5. Enable GitHub Pages in repository settings (Source: Deploy from Branch, select main branch)
6. Your site will be live at `https://flyingbluekite.github.io`

## Adding Instagram Photos
Replace placeholder image URLs in `about.html` with your actual Instagram photo URLs.

## Writing New Posts
Create new .md files in `_posts/` folder with format: `YYYY-MM-DD-title.md`
